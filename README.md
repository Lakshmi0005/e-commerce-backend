# E-Commerce Backend

Basic Express.js backend with:

- User CRUD
- Product CRUD
- Order Creation
- MongoDB Database
- Layered Architecture:
  - Models
  - Services
  - Handlers
  - Routes

## Endpoints

### Users
POST /users
GET /users
PUT /users/:id
DELETE /users/:id

### Products
POST /products
GET /products
PUT /products/:id
DELETE /products/:id

### Orders
POST /orders
GET /orders
